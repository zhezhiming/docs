---
title: 日历控制器
sidebar_position: 1
---

从输入控制器属性设置界面的主设置框里的 "**维度**" 中选择一个 **日历** 类型的维度, 那么输入控制器自动使用时间组件来展示输入控制.
系统会自动根据日历维度的时间粒度属性(或者在属性设置里手动选择时间粒度)来展示不同的时间组件类型.
常见的时间粒度有: 年,季度,月,周,日 等.

选择时间后组件会根据日历维度配置的日期格式化格式将时间转换成日历维度成员关键值.

其他日历相关属性:

| 名称             | 说明                                                           |
| ---------------- | -------------------------------------------------------------- |
| 类型         | 时间选择的类型, 有 **单值** 和 **区间**                                     |
| 时间粒度 | 当前时间组件默认的时间粒度, 系统会根据此时间粒度找到维度上对应的时间粒度层级上的格式化字符串, 将当前选中的时间格式化成响应的字符串值. 关于模型的日历设置参考[日历维度](../../../models/mdx-modeling/calendar-dimension)或者[为第三方多维模型创建语义增强](../../../models/mdx-modeling/semantic-enhancement-for-third-party-olap-cube) |
| 时间粒度序列 | 除了默认的时间粒度还可以设置可选的其他时间粒度即此属性一个时间粒度序列 |
| 时间格式 | 对当前时间进行格式化成维度成员, 如果不设置则系统会从维度相应的层级中找到时间格式, 再否则使用默认的格式化格式 |
| 默认日期函数 | 默认显示的日期函数, 函数会根据当前时间设置计算出相应的事件, 进而决定了按时间粒度格式化后的日历维度值 |
| 默认日期 | 选择了默认日期函数后, 此属性会显示相应的函数名, 如果想要指定某个具体固定日期可以按格式 `yyyy-MM-dd`进行填写一个日期如: `2022-06-06` |
