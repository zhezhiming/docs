**参数控制器** 是在输入控制器中为多维数据集的参数进行输入控制的组件.

:::tip

关于参数的创建详见 [参数](../../../stories/parameter/).

:::

## 输入

**输入** 类型的参数对应的控制器, 基础形式是一个输入框, 允许用户手动输入值. 如果参数类型为 **Number** 还可以选择使用滑杆来控制参数值, 滑杆详细参数如下:

| 名称         | 说明                         |
| ------------ | ---------------------------- |
| 使用滑杆 | 是否使用滑杆组件展示参数值的输入 |
| 滑杆最大值 | 滑杆允许的最大值 |
| 滑杆最小值 | 滑杆允许的最小值 |
| 滑杆单步大小 | 滑杆滑动一步所产生的值的变化大小 |
| 滑杆间隔长度 | 滑杆刻度间隔实际展示长度 |
| 滑杆倒置 | 反向展示滑杆 |
| 滑杆垂直 | 垂直展示滑杆 |

## 选择

**选择** 类型的参数将会展示为可选值的选择列表. 可选列表成员将使用参数里预输入的值列表. 目前支持单选.

## 维度成员

**维度** 类型的参数以给定的维度为基础, 展示预先选择好的成员作为控制器的可选项, 用户从这些成员中选择一个或多个作为结果值.
