平面表格是以**维度**为**行**配置, **度量**为**列**配置的表格形式.

## 数据配置

平面表格的数据配置主要是以维度为行, 度量为列, 不涉及以维度成员为列和维度展示为树型结构的情况(那将是[数据透视表](./../pivot-grid/)的功能).

## 表格配置

除了通用的分析表格树型外, 平面表格特有的配置选项还有:

| 属性         | 英文                   | 描述                           |
| ------------ | ---------------------- | ------------------------------ |
| 分页   | **Paging**       | 是否分页显示   |
| 每页行数   | **Page Size**       | 分页的每页表行数   |
