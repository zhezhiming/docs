文本组件是指在仪表板上添加的用于展示文本信息的组件。它允许用户在仪表板上添加标题、段落、说明或其他文本内容，以提供更多的上下文和解释，帮助用户理解仪表板中呈现的数据和图表。

在故事编辑界面, 点击工具栏中插入区域更多里的"**文本**", 在当前页面中创建一个文本组件. 选中此组件后在右侧属性编辑栏中对其设置进行编辑:

## 构建

文本微件属性如下：

| 属性      | 描述 |
| ----------- | ----------- |
| 文本 | 输入文本组件的文字内容，也可以在文本组件上进行编写。 |
| **导航** |  |
| 导航类型 | 故事点：转向另一个故事点页面 |
| 页面 | 故事点页面选择 |
