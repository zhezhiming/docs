---
title: 下拉选择器
sidebar_position: 3
---


下拉选择器将维度成员或者用户自定义成员展示成下拉列表供用户选择. 用户可以选择一个或多个成员.

:::tip 提示

下拉选择器一般用于可选成员较少的情况.

:::

详细配置:

| 名称         | 说明                                                     | 详细信息 |
| ------------ | -------------------------------------------------------- | -------- |
| 标签         | 输入组件的标签                                           |          |
| 占位符       | 输入组件的占位符                                         |          |
| 多选         | 维度成员可以多选                                         |          |
| 最大标签数量 | 多选时组件上展示出选中成员的最大数量, 其余用省略符号代替 |          |
| 用户自定义值 | 用户自定义成员选项                                       |          |
