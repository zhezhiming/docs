---
title: 日历选择器
sidebar_position: 4
---

**日历选择器** 通常用于分析时间序列数据，如销售额、产量等。日历选择器可以以日历的形式显示出来，用户可以选择日期范围，如最近一个月、上个季度等。这样可以更好地查看时间序列数据，并帮助用户进行分析和决策。

如果维度的语义类型为日历, 那么组件将成员选择器展示成日历组件, 用户选择相应的时间, 组件会将用户选择的时间转换成相应的成员值. 如果不想将日历类型的维度展示成日历组件, 可以将属性 **控制器类型** 设置为其他类型, 过滤器栏组件将其展示成相应的选择器组件.

日历选择器的属性列表:

| 名称         | 说明                                                    | 详细信息 |
| ------------ | ------------------------------------------------------- | -------- |
| 标签         | 输入组件的标签                                          |          |
| 占位符       | 输入组件的占位符                                        |          |
| 时间粒度     | 年,季度,月,日等时间粒度, 选择一个作为组件默认的时间粒度 |          |
| 时间粒度序列 | 组件可选的时间粒度序列                                  |          |
| 时间格式     | 组件输出时间成员值的格式                                |          |
| 默认时间函数 | 默认时间的函数                                          |          |
| 默认时间     | 默认的固定时间                                          |          |
