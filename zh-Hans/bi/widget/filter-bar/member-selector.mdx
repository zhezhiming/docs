---
title: 成员选择器
sidebar_position: 2
---

**维度成员选择器** 可以让用户根据维度来筛选数据，如时间、地区、产品类型等。这样可以更好地查看不同维度的数据，并帮助用户进行分析和决策。成员选择器以下拉列表的方式平铺展示维度成员共用户搜索和选择, 用户还可以点击帮助按钮打开维度成员选择帮助框进行更详细的选取.

成员选择器属性列表:

| 名称             | 说明                                                     | 详细信息 |
| ---------------- | -------------------------------------------------------- | -------- |
| 标签             | 输入组件的标签                                           |          |
| 占位符           | 输入组件的占位符                                         |          |
| 自动激活第一选项 |                                                          |          |
| 多选             | 维度成员可以多选                                         |          |
| 最大标签数量     | 多选时组件上展示出选中成员的最大数量, 其余用省略符号代替 |          |
| 可搜索           | 组件具有成员选项搜索功能                                 |          |
