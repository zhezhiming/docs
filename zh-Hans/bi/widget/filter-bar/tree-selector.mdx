---
title: 树型选择器
sidebar_position: 1
---

**树型选择器** 是 BI 报表中的一种常见元素，它允许用户在一个层次结构的界面中选择数据。通常，树型选择器用于组织和选择大量的数据，如组织架构、地理位置、产品类别等。树型选择器由一个根节点和多个子节点组成，用户可以通过展开和折叠节点来筛选数据。这种方式能够帮助用户更快捷地找到所需数据，并使报表更直观和易于理解。

树型选择器属性列表:

| 名称         | 说明                                                     | 详细信息                                                                                             |
| ------------ | -------------------------------------------------------- | ---------------------------------------------------------------------------------------------------- |
| 标签         | 输入组件的标签                                           |                                                                                                      |
| 占位符       | 输入组件的占位符                                         |                                                                                                      |
| 初始层级     | 初始展示的层级深度                                       |                                                                                                      |
| 自动补全     | 输入关键字过滤相关成员                                   |                                                                                                      |
| 可搜索       | 组件具有成员选项搜索功能                                 | 如果选择可搜索, 组件会使用自动补全功能, 自动补全再叠加**虚拟滚动**功能, 那么可选项面板会使用虚拟滚动 |
| 虚拟滚动     | 启用成员选项的虚拟滚动                                   | 暂时只有**可搜索**条件下才有虚拟滚动功能                                                             |
| 多选         | 维度成员可以多选                                         |                                                                                                      |
| 最大标签数量 | 多选时组件上展示出选中成员的最大数量, 其余用省略符号代替 |                                                                                                      |
| 面板宽度     | **自动补全**情况下                                       | 下拉面板宽度自定义设置                                                                               |
