**首选项** 菜单里可以配置故事的常规属性和样式。

![故事仪表板首选项](/public/img/story/story-preferences.png)

菜单中按顺序依次为：

| 名称           | 链接         | 描述                                                           |
| -------------- | ------------ | -------------------------------------------------------------- |
| 打开模型      | [语义模型](../../models/)       | 转到故事仪表板所使用的语义模型工作空间     |
| 新页面      | [新画布页面](../workspace/#新画布页面)       | 打开创建新页面     |
| 故事详情 | [故事详情](../story-details/) | 打开故事详情窗口 |
| 高级样式 | [高级样式](../advanced-styles/) | 打开高级样式编辑窗口 |
| 计算成员 | [管理计算度量和参数](../calculations/#管理计算度量和参数) | 打开故事的计算成员和参数编辑窗口 |
| 主题构建器 |  | 打开三种（Light, Dark, Thin）主题的配置编辑器窗口 |
| 模版 | [模版库](../template/) | 打开故事模版库 |
| 编辑 |  | 打开编辑子菜单，功能包括：保持，复制，粘贴，另存为模版 |
| 分享 | [分享](../share/) | 打开故事分享窗口 |
| 导出 |  | 导出故事仪表板为本地文件 |
| 平移 | [平移](#平移) | 切换到可平移画布的模式 |
| 缩放 | [缩放](#缩放) | 缩放画布大小 |

## 平移

平移模式下，可以通过鼠标拖拽的方式平移画布。用户可以按住 `Alt` 键，然后用鼠标拖动画布。

## 缩放

缩放模式下，可以通过点击放大缩小按钮对画布进行缩放，也可以直接使用菜单中的缩放级别直接缩放。也可以使用快捷键：

1. **Alt** + **+** 放大 10%
2. **Alt** + **-** 缩小 10%
3. **Alt** + **Esc** 恢复初始状态

## 快捷键

| 快捷键 | 描述 |
| --- | --- |
| `Ctrl + S` | 保存当前故事 |
| `Ctrl + Z` | 撤销 |
| `Ctrl + Shift + Z` | 重做 |
| `Alt + -` / `Alt + Wheel down` | 缩小 |
| `Alt + (=/+)` / `Alt + Wheel up`| 放大 |
| `Alt + 0` | 重置缩放 |
| `Alt + Pointer` | 平移 |
| `Alt + Esc` | 重置缩放和平移 |
