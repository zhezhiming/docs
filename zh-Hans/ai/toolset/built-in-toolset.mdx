内置工具提供了开箱即用的能力，You can directly use the first-party built-in tools provided by the Xpert ecosystem.

## 如何配置内置工具？

只需要在工作空间中选择某个内置工具集，并添加授权信息和开启需要的工具后便可以在智能体中使用。用户还可以点击测试按钮进行手工测试工具。

![Builtin Toolset authorization](/public/img/ai/xpert-toolset-builtin-auth.png)

内置工具集目前包括以下：

| 工具名称         | 描述                                       | 状态     |
|-----------------|--------------------------------------------|---------|
| **TavilySearch**| 专注于特定领域的信息检索工具。               | 启用     |
| **DuckDuckGo**  | 提供隐私保护的搜索引擎，支持网页搜索。       | 开发中     |
| **Wikipedia**   | 访问维基百科，获取丰富的百科全书信息。       | 开发中     |
| **SearchApi**   | 通用搜索接口，整合多种搜索服务。             | 开发中     |
| **ExaSearch**   | 提供多种搜索引擎的整合和比较。               | 开发中     |
| **SearxngSearch**| 开源的聚合搜索引擎，支持多种搜索引擎查询。  | 开发中     |
| **BingSearch**| Bing 搜索引擎，支持搜索引擎查询。  | 开发中     |

## 数据分析平台工具集

除了常用的工具集外，Xpert 还提供了与其数据分析平台无缝集成的分析类工具集：

| 工具名称         | 描述                                       | 状态     |
|-----------------|--------------------------------------------|---------|
| **ChatBI**  | 与语义模型对话分析和展示的 ChatBI 工具集。       | 启用     |
| **ChatDB**   | 与数据库表直接对话分析数据的 ChatDB 工具集。       | 启用     |

详见 [数据分析类工具集](../data/).
