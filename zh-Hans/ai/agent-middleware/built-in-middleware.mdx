# 内置中间件

## 概述

内置中间件提供了系统内置的中间件功能，开箱即用，无需额外配置。

## 功能特性

- **请求拦截**: 拦截和处理请求
- **日志记录**: 记录执行日志
- **错误处理**: 统一错误处理
- **性能监控**: 监控执行性能
- **认证授权**: 处理认证和授权

## 内置中间件类型

### 日志中间件

- **请求日志**: 记录所有请求
- **响应日志**: 记录所有响应
- **错误日志**: 记录错误信息
- **性能日志**: 记录性能指标

### 认证中间件

- **Token验证**: 验证访问Token
- **权限检查**: 检查用户权限
- **会话管理**: 管理用户会话

### 错误处理中间件

- **错误捕获**: 捕获执行错误
- **错误格式化**: 格式化错误信息
- **错误通知**: 发送错误通知

## 使用方法

1. 进入智能体中间件页面
2. 选择内置中间件
3. 配置中间件参数
4. 启用中间件
5. 应用到工作流
