在完成知识库创建后，你可以继续对其进行管理与优化，以确保它能为智能体或应用提供更精准的上下文支持。
XpertAI 支持在知识流水线发布后，通过多种方式对知识库进行维护与改进：

### 管理流水线

你可以直接浏览并编辑知识流水线的编排结构，调整节点配置或更新文档转换策略，从而优化知识的处理与召回效果。

### 知识库维护

在知识库页面，你可以：

* **更新文件内容**：重新上传或替换知识文件；
* **进行召回测试**：验证知识召回效果，评估问答准确度；
* **修改流水线**：回到知识流水线编辑界面进行优化；
* **编辑知识库属性**：包括知识库名称、描述、标签等基础信息。

了解更多内容，请前往 [知识库与文档管理](/docs/ai/knowledge/documents)。
