---
title: 步骤三：发布知识流水线
sidebar_position: 3
---

当你完成知识流水线的编排与调试后，即可将其正式发布。
在 XpertAI 中，**知识流水线（Knowledge Pipeline）是数字专家（Digital Expert）的一种能力扩展**，因此其发布流程与数字专家的发布机制完全一致。

点击页面右上角的 **「发布」** 按钮，在弹窗中输入发布说明确认后，即成功发布知识流水线。

成功发布后，你可以进行以下操作：

### 🧩 添加文档

点击此选项后，你将跳转至知识库的数据源配置界面，可以直接上传文档或通过第三方数据源（如飞书文档、网页抓取等）进行导入。

### 🔗 访问 API

前往 API 文档页面，可查看该知识流水线的 API 调用方式、输入输出参数以及示例代码，便于外部系统集成调用。

### 🧠 下载为知识流水线模板

你可以将当前流水线下载为一个可复用的模板。
模板将在自定义模板列表中展示（开发中），供其他知识库快速复用相同的知识处理逻辑。
