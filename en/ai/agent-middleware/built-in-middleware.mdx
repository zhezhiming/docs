# Built-in Middleware

## Overview

Built-in middleware provides system-built middleware functionality that works out of the box without additional configuration.

## Features

- **Request Interception**: Intercept and process requests
- **Logging**: Record execution logs
- **Error Handling**: Unified error handling
- **Performance Monitoring**: Monitor execution performance
- **Authentication & Authorization**: Handle authentication and authorization

## Built-in Middleware Types

### Logging Middleware

- **Request Logs**: Record all requests
- **Response Logs**: Record all responses
- **Error Logs**: Record error information
- **Performance Logs**: Record performance metrics

### Authentication Middleware

- **Token Verification**: Verify access tokens
- **Permission Check**: Check user permissions
- **Session Management**: Manage user sessions

### Error Handling Middleware

- **Error Capture**: Capture execution errors
- **Error Formatting**: Format error information
- **Error Notification**: Send error notifications

## Usage

1. Navigate to the Agent Middleware page
2. Select Built-in Middleware
3. Configure middleware parameters
4. Enable the middleware
5. Apply to workflow
