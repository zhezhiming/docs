The Monitor Dashboard page provides features for querying and viewing monitor logs.

At the top are the entry points for **Embedded Page Applications** and **Backend Service APIs**.

Monitor analysis features include:  
- Total number of conversations  
- Number of active users  
- Average session interaction count  
- Total number of messages  
- User satisfaction  
- Token response speed  
- Total token consumption
