title: Quick Access Application
sidebar_position: 3
---

Customers can configure the ChatBI webpage as a web application feature of their enterprise's custom-built application, allowing ChatBI to be used in the Quick Access menu within the Feishu chat application. After configuring the ChatBI web application in the Feishu Admin Console, you can add the ChatBI application to the Quick Access menu in the Feishu chat app.

![Lark chatbi webpage](/public/img/chatbi/lark-chatbi-webpage.png)

By using the ChatBI Quick Access application, you can directly open the ChatBI webpage in the chat window sidebar and perform data analysis. The ChatBI webpage offers more comprehensive features and analysis capabilities compared to the Feishu bot.

:::tip
Using the ChatBI web version allows users to achieve professional data analysis capabilities.
:::
