---
title: Key Indicator
sidebar_position: 5
---

The **Key Indicator** component is typically used to display important business data, such as sales revenue, customer count, conversion rate, etc. They are usually presented in a concise form, including numbers and icons, to facilitate quick reading and understanding of information.

Key indicators can obtain indicator values through the **Value** field, and can also add a **Target Value** field to compare the difference between the value and the target value.

The key indicator component consists of one primary indicator and multiple other data indicators. If you want to list detailed multiple indicators, you can set other data indicators.

| Property | Description |
| ----------- | ----------- |
| Language Model | Select language model |
| Model Entity | Select an entity "Multidimensional Dataset" from the language model |
| Title | Title displayed for the indicator |
| Value | Measure field of the indicator |
| Target Value | Target value measure field of the indicator |
| Other Data Indicators | Add multiple other data indicators with the same properties as the primary indicator |
| **Properties** | |
| Abbreviate Numbers | Whether to abbreviate indicator value number format |
| Number Format | Indicator number display format, e.g., 0.0-2 represents a number format with thousand separators, minimum 0 decimal places, maximum 2 decimal places |
| Unit | Unit of the indicator |
| Unit Semantic | Semantic type of indicator unit: currency code, unit of measure... |
| Show Deviation | Whether to show the deviation percentage between indicator value and target value |
| Deviation Text | Text description of the displayed indicator deviation |
| **Navigation** | |
| Navigation Type | Story Point: Navigate to another story point page |
| Page | Story point page selection |
