The iframe component uses the [iframe](https://developer.mozilla.org/en-US/docs/Web/HTML/Element/iframe) element to display embedded pages.

In the story editing interface, click "**iframe**" in the toolbar's insert area to create an iframe component on the current page. After selecting this component, edit its settings in the property editing panel on the right:

> Since the iframe embeds web page content, clicking on the component interacts with the embedded page. To select the component itself, click the drag bar on the edge of the component. After selecting, click the menu button in the upper right corner to activate the component editing interface.

## Build

Build settings have one property:

* **Source URL**: Enter the web page address to embed;

## Style

Style includes general component style configuration.
