---
title: Member Selector
sidebar_position: 2
---

**Dimension Member Selector** allows users to filter data by dimensions, such as time, region, product type, etc. This enables better viewing of data across different dimensions and helps users with analysis and decision-making. Member selectors display dimension members in a flat dropdown list for user search and selection. Users can also click the help button to open the dimension member selection help box for more detailed selection.

Member selector property list:

| Name | Description | Details |
| ---------------- | -------------------------------------------------------- | -------- |
| Label | Input component label | |
| Placeholder | Input component placeholder | |
| Auto Activate First Option | | |
| Multiple Selection | Dimension members can be multi-selected | |
| Max Label Count | Maximum number of selected members displayed on the component in multi-select mode, others replaced with ellipsis | |
| Searchable | Component has member option search functionality | |
