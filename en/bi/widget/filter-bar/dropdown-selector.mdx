---
title: Dropdown Selector
sidebar_position: 3
---


Dropdown selectors display dimension members or user-defined members as a dropdown list for user selection. Users can select one or more members.

:::tip Tip

Dropdown selectors are generally used when there are fewer selectable members.

:::

Detailed configuration:

| Name | Description | Details |
| ------------ | -------------------------------------------------------- | -------- |
| Label | Input component label | |
| Placeholder | Input component placeholder | |
| Multiple Selection | Dimension members can be multi-selected | |
| Max Label Count | Maximum number of selected members displayed on the component in multi-select mode, others replaced with ellipsis | |
| User Custom Values | User-defined member options | |
