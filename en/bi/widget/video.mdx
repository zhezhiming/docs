The video widget allows users to embed video content in dashboards, providing a more vivid and diverse way of presenting information.

Property list:

| Property | Description |
| ----------- | ----------- |
| Video Link | Link to the video file |

:::tip
If the link is from a video website, you usually need to use the [iFrame Widget](../iframe/) to embed it into the dashboard.
:::
