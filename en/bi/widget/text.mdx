The text component is a component added to the dashboard for displaying text information. It allows users to add titles, paragraphs, descriptions, or other text content on the dashboard to provide more context and explanations, helping users understand the data and charts presented in the dashboard.

In the story editing interface, click "**Text**" in the toolbar's insert area to create a text component on the current page. After selecting this component, edit its settings in the property editing panel on the right:

## Build

Text widget properties are as follows:

| Property | Description |
| ----------- | ----------- |
| Text | Enter the text content of the text component, or write directly on the text component. |
| **Navigation** |  |
| Navigation Type | Story Point: Navigate to another story point page |
| Page | Story point page selection |

Style configuration includes general component style settings.
