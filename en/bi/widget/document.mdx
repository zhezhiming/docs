The document component uses the [TinyMCE](https://github.com/tinymce/tinymce) library for editing and displaying rich text online.

In the story editing interface, click "**Document**" in the toolbar's insert area to create a document (rich text) component on the current page. After selecting this component, edit its settings in the property editing panel on the right:

## Data Configuration

Configure **data source** and **model** for calculating indicator results inserted into the document.

Insert indicators from the configured model into the document using the format `{{[#INDICATOR_CODE]}}`. When previewing or displaying, the system will automatically calculate the corresponding data results and display them in the appropriate location.

## Style

Style configuration includes general component style settings.
