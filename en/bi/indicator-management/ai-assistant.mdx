List of Copilot commands for metrics:

| Command | Description |
| --- | --- |
| `/i` | Create metric |
| `/formula` | Create derived metric formula |

## /i

Command to create metrics, Copilot will automatically create metrics based on the description provided in the command.

## /formula

When creating a command for derived metric formulas, it's important to describe the formula's meaning as clearly as possible so that Copilot can better understand it.
