# 飞书登录

## 概述

飞书登录功能允许用户通过飞书账号进行单点登录，实现统一的身份认证和访问控制。

## 功能特性

- **飞书集成**: 与飞书平台集成，支持飞书账号登录
- **自动同步**: 自动同步飞书用户信息
- **权限映射**: 支持飞书角色到系统角色的映射
- **安全认证**: 使用OAuth 2.0协议进行安全认证

## 配置步骤

### 1. 在飞书开放平台创建应用

1. 登录飞书开放平台
2. 创建企业自建应用
3. 获取应用凭证
   - App ID
   - App Secret
4. 配置回调地址

### 2. 在平台中配置飞书登录

1. 进入网站功能页面
2. 点击"单点登录" -> "飞书登录"
3. 填写配置信息
   - **App ID**: 飞书应用的App ID
   - **App Secret**: 飞书应用的App Secret
   - **回调地址**: OAuth回调地址
4. 配置用户属性映射
   - 用户名映射
   - 邮箱映射
   - 角色映射
5. 测试连接
6. 保存配置

## 配置选项

### 应用配置

- **App ID**: 飞书应用的唯一标识
- **App Secret**: 飞书应用的密钥
- **回调URL**: OAuth回调地址

### 用户属性映射

- **用户名**: 映射飞书用户名到系统用户名
- **邮箱**: 映射飞书邮箱到系统邮箱
- **显示名称**: 映射飞书显示名称
- **角色**: 映射飞书角色到系统角色

### 权限设置

- **自动创建用户**: 首次登录时自动创建用户
- **自动更新用户**: 自动更新用户信息
- **默认角色**: 设置新用户的默认角色

## 使用说明

### 用户登录

1. 用户点击"飞书登录"按钮
2. 跳转到飞书授权页面
3. 用户授权后跳转回平台
4. 自动完成登录

### 用户同步

- 首次登录时自动创建用户账号
- 后续登录时自动更新用户信息
- 支持角色和权限的自动同步

## 注意事项

- 确保回调地址配置正确
- 定期更新App Secret
- 注意用户权限的映射关系
- 测试时使用测试环境的应用凭证
