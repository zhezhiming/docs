---
title: "ClickHouse"
description: "ClickHouse列式数据库连接配置"
---
On the data source management page, click **New Data Source**.

Select the data source type: **ClickHouse**, and then configure the following details:

| Properties | English | Description |
| -------- | ----------------- | -------------------------------------- |
| Link | **Url** | Database link (or configure the host and port number separately) |
| Host | **Host** | Host address domain name or IP |
| Port number | **Port** | Service port number |
| User | **User Name** | Account used to connect to the data source |
| Password | **Password** | Account and password used to connect to the data source |
| Database Name | **Database Name** | Database instance to connect to the data source |
