## Create Hive data source

Click **New Data Source** in the data source management interface, select **Hive** type in the pop-up interface, and enter Hive connection information:

* Name: unique name of the data source;

* **Use local proxy**:

* Host: server domain name or IP address;

* Port: port number of HiveServer2 service;

* Other fields TODO

Click Save, creation is successful.
