---
title: "MariaDB"
description: "MariaDB关系型数据库连接配置"
---
Select the data source type: **MariaDB**, and then configure the following details:

| Properties | English | Description |
| -------------- | ------------------- | -------------------------------------------------------- |
| Name | **Name** | Data source name, keep it unique |
| Use local agent | **Use Local Agent** | Whether to use the local agent to connect to the data source, otherwise use the server to connect to the data source |
| Host | **Host** | Host address domain name or IP |
| Port number | **Port** | Service port number |
| User name | **User Name** | Account used to connect to the data source |
| Password | **Password** | Account password used to connect to the data source |
| Use encrypted connection | **Use SSL** | Whether to use encrypted connection |
| CA certificate | **CA certificate** | Client certificate |
