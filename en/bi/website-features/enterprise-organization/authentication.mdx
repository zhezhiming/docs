# Authentication

## Overview

Authentication functionality is used to manage authentication configuration for enterprise organizations, including identity authentication, permission authentication, etc.

## Features

- **Identity Authentication**: User identity verification
- **Permission Authentication**: Permission verification and authorization
- **Authentication Configuration**: Configure authentication methods
- **Security Settings**: Security authentication settings

## Authentication Methods

### Password Authentication

- **Password Policy**: Set password policy
- **Password Complexity**: Configure password complexity requirements
- **Password Expiration**: Set password expiration policy

### Single Sign-On

- **SSO Configuration**: Configure single sign-on
- **Third-Party Authentication**: Integrate third-party authentication services

### Two-Factor Authentication

- **2FA Enablement**: Enable two-factor authentication
- **Authenticator Configuration**: Configure authenticator application

## Configuration Steps

1. Navigate to the Authentication page
2. Select authentication method
3. Configure authentication parameters
4. Test authentication
5. Save configuration
