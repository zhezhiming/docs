# Function Module

## Overview

Function module management is used to configure available function modules for tenants and control tenant function access permissions.

## Function Module List

### Data Analysis Modules

- Semantic Model
- Indicator Management
- Story Dashboard

### Widget Modules

- Analysis Card
- Analysis Table
- Filter Bar
- Input Controller

### Website Feature Modules

- Data Source Management
- Single Sign-On
- Enterprise Tenant Management

## Configuration Method

1. Navigate to the Function Module page
2. Select tenant
3. Enable/disable function modules
4. Set module permissions
5. Save configuration
