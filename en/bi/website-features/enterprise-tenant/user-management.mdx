# User Management

## Overview

User management is used to manage users within a tenant, including user creation, editing, deletion, and permission assignment.

## User Operations

### Adding Users

1. Click the "Add User" button
2. Fill in user information
   - Username
   - Email
   - Display Name
   - Password
3. Assign role
4. Assign organization
5. Save user

### Editing Users

1. Select the user to edit
2. Modify user information
3. Update role and organization
4. Save changes

### Deleting Users

1. Select the user to delete
2. Click the "Delete" button
3. Confirm deletion operation

## User Information

- **Username**: User's login name
- **Email**: User's email address
- **Display Name**: User's display name
- **Role**: Role the user belongs to
- **Organization**: Organization the user belongs to
