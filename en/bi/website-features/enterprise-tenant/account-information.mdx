# Account Information

## Overview

Account information is used to manage basic information and security settings of user accounts.

## Account Information

### Basic Information

- **Username**: User's login name
- **Email**: User's email address
- **Display Name**: User's display name
- **Avatar**: User avatar

### Security Settings

- **Password**: Change password
- **Two-Factor Authentication**: Enable/disable two-factor authentication
- **Login History**: View login history records

## Modifying Account Information

1. Navigate to the Account Information page
2. Modify information that needs to be updated
3. Save changes
