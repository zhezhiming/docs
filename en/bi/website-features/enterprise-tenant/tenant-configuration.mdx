# Tenant Configuration

## Overview

Tenant configuration is used to manage tenant basic information and settings, including tenant name, status, resource quotas, etc.

## Configuration Items

### Basic Information

- **Tenant Name**: Display name of the tenant
- **Tenant Code**: Unique identifier of the tenant
- **Tenant Description**: Detailed description of the tenant
- **Tenant Status**: Enabled/disabled status

### Resource Quotas

- **Storage Space**: Data storage space limit
- **Computing Resources**: Computing resource limit
- **User Count**: User count limit
- **API Calls**: API call count limit

### Function Settings

- **Enabled Functions**: Select function modules available to the tenant
- **Function Limits**: Set function usage limits

## Usage Method

1. Navigate to the Enterprise Tenant page
2. Select the tenant to configure
3. Click "Tenant Configuration"
4. Modify configuration items
5. Save configuration
