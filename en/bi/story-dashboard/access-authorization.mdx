Dashboard permissions are as follows:

- **Public**: Everyone can view and access the dashboard.
- **Private**: Only creators and administrators can view and access the dashboard.
- **Belonging to Business Area**: Users with permissions in the business area can view and access all dashboards within this business area.
- **Belonging to Project**: Project members can view and access all dashboards within the project.

Editing permissions:

- **Creator**: Can edit and delete the dashboard.
- **Administrator**: Can edit and delete the dashboard.
- **Project Member**: Can edit dashboards within the project.
